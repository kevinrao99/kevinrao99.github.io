Tree of Life

placeholder

One of my hobbies is learning about nature and the evolutionary relationship between species. I started this personal project as a tool to help record interesting facts as I learn them, and over time the tree has kept growing as I keep adding to it. However, it is far from complete, as which branches grow just depends on what I've been interested in so far. The actual version is in Overleaf and this page is updated by periodic pdf uploads.

When the tree has gotten too deep and is continued on a different pdf, this is denoted with the "∈" symbol (I just picked something that stands out). From top to bottom, the pdfs are arranged in some valid topological sort order. Green highlighting denotes information about evolutionary events relative to geological eras, and yellow highlighting denotes particularly fascinating anatomical ideas.

Each pdf has a lot of white space at the bottom because in Overleaf I set the paperheight to be 100cm to accomodate the tree's growth and avoid any file from having awkward page breaks in the middle. If you happen to know how to get Latex to automatically adjust page length to fit the contents, please let me know!
